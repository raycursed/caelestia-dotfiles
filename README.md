# caelestia-dotfiles
custom configs and keybinds

```bash
git clone https://github.com/raycursed/caelestia-dotfiles.git ~/caelestia-dotfiles
```


```bash
cp -r ~/caelestia-dotfiles/* ~/.config/hypr/
```


```bash
hyprctl reload
```

## Keybinds
Super + Enter                  -> Terminal (foot)
Super + W                      -> Browser (zen-browser)

Super + Space                  -> Special workspace
Super + Shift + 1,2,3,4...0    -> Move window to workspace
Super + Shift + Space          -> Move window to Special workspace
